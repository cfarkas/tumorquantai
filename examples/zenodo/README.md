# Zenodo release templates

These templates are intentionally incomplete and fail closed.

- Copy `metadata.template.json` to a protected working directory and replace
  every required blank with approved Zenodo metadata.
- Copy `authorization.template.json` only after the final upload plan and draft
  have been reviewed. An accountable authorized person must complete it.
- Keep working copies, tokens, private mappings, state files, and review evidence
  outside the repository and public artifact directory.
- Never change confirmation values to `true` merely to make a command pass.

See [the publishing guide](../../docs/ZENODO_PUBLISHING.md).
